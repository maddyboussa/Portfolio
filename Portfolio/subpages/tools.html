CURRENT WORK IN PROGRESS

In my studies of OpenGL and C++ graphics programming, I have created multiple small scale applications that handle the computation and rendering of objects on screen.

Polygon Creation Tool

Requiring an in-depth understanding of the graphics pipeline, I have developed an application that draws user-generated polygons to the screen and provides basic user controls of color, placement, and dynamic drawing / creation. In order to do this, I utilized a deep understanding of the rasterization process, storage of vertex data and transfer of array data from the CPU to GPU for rendering, as well as knowledge of OpenGL syntax and callback functions. With an emphasis on memory and performance efficiency, a class structure was implemented to handle polygon creation.

Particle System

I developed a particle system simulator using C++ and OpenGL to dynamically create and render particles to the screen in a performance efficient way. In order to do so, I implemented a Particle System class that handles the vertex data including velocity, position, and alpha over lifetime. Particle vertex data is sent to the GPU using vertex array drawing methods in order to reduce the processing power required to simulate many objects. A camera class is also created to allow users to alter the view angle of the particle system and visualize the coordinate system, as well as handling the projection of 3D objects onto a 2D viewing plane.
